Maquettes Interface Graphique

Utilisateur :
Samir Crypto

Objectif

Créer une interface graphique permettant de gérer :

    -les ligues
    -les employés
    -les administrateurs
    -les comptes utilisateurs

1. Fenêtre de connexion
Contenu
    /Champ Login
    /Champ Mot de passe
    /Bouton Se connecter
    /Bouton Quitter

    Message d'erreur en cas d'identifiants incorrects

Fonction: Permet à un utilisateur de se connecter :

    ROOT
    Administrateur de ligue
    Employé

Après authentification, l'utilisateur est redirigé vers son espace correspondant.

2. Fenêtre principale
Contenu
    /Bouton Gestion des ligues
    /Bouton Gestion des employés
    /Bouton Définir administrateur
    /Bouton Mon profil
    /Bouton Déconnexion
    /Bouton Quitter

Fonction: Fenêtre centrale permettant d'accéder aux différentes fonctionnalités de l'application.

3. Fenêtre Liste des ligues
Contenu

Tableau des ligues :

    Numéro de ligue
    Nom de la ligue
    Administrateur

    Actions
    Bouton Ajouter ligue
    Bouton Modifier ligue
    Bouton Supprimer ligue
    Bouton Retour

Fonction: Afficher l'ensemble des ligues enregistrées.

4. Fenêtre Ajouter / Modifier une ligue
Contenu
    Champ Nom de la ligue
    Actions
    Bouton Valider
    Bouton Annuler
Fonction:
    Ajout

Le champ est vide.

    Modification

Le champ est pré-rempli avec les informations de la ligue sélectionnée.

5. Fenêtre Gestion des employés
Contenu

    Liste des employés :

    Identifiant
    Nom
    Prénom
    Mail
    Ligue
    Actions
        Bouton Ajouter employé
        Bouton Modifier employé
        Bouton Supprimer employé
        Bouton Retour
Fonction: Permet de gérer les employés d'une ligue.

6. Fenêtre Liste des employés
Contenu

Pour chaque employé :

    Numéro employé
    Nom
    Prénom
    Mail
    Date d'arrivée
    Date de départ
    Ligue associée
    Actions
        Bouton Ajouter
        Bouton Modifier
        Bouton Supprimer
        Bouton Retour
Fonction: Afficher l'ensemble des employés enregistrés.

7. Fenêtre Ajouter / Modifier un employé
Contenu
    Champ Nom
    Champ Prénom
    Champ Mail
    Champ Mot de passe
    Champ Date d'arrivée
    Champ Date de départ
    Liste déroulante des ligues
    Actions
        Bouton Valider
        Bouton Annuler
        Fonction
        Ajout
        Date d'arrivée remplie automatiquement avec la date du jour.
        Date de départ facultative.
        Modification

Les champs sont pré-remplis avec les informations de l'employé.

8. Fenêtre Définir administrateur
Contenu

Liste des employés :

    Numéro employé
    Nom
    Prénom
    Mail
    Ligue
    Actions
    Bouton Définir administrateur
    Bouton Retirer administrateur
    Bouton Retour
Fonction: Associer un employé comme administrateur d'une ligue.

9. Fenêtre Profil utilisateur
Contenu
    Nom
    Prénom
    Mail
    Ligue
    Date d'arrivée
    Date de départ
    Actions
    Bouton Modifier mot de passe
    Bouton Retour
Fonction: Permet à un utilisateur de consulter ses informations personnelles.

10. Gestion du ROOT
Accès

Accessible uniquement au compte ROOT.

    Le ROOT peut
    Ajouter une ligue
    Modifier une ligue
    Supprimer une ligue
    Ajouter un employé
    Modifier un employé
    Supprimer un employé
    Définir un administrateur
    Retirer un administrateur
    Consulter toutes les ligues
    Consulter tous les employés
11. Gestion de l'administrateur de ligue
Accès

Accessible uniquement aux employés désignés administrateurs.

    L'administrateur peut
    Consulter sa ligue
    Consulter les employés de sa ligue
    Ajouter un employé dans sa ligue
    Modifier un employé de sa ligue
    Supprimer un employé de sa ligue
    Restrictions
    Ne peut pas supprimer une ligue
    Ne peut pas créer une ligue
    Ne peut pas définir un autre administrateur
12. Gestion de l'employé
Accès

Accessible à tous les employés.

    L'employé peut
    Se connecter
    Consulter son profil
    Modifier son mot de passe
    Restrictions
    Ne peut pas modifier les ligues
    Ne peut pas gérer les employés
    Ne peut pas définir d'administrateur
